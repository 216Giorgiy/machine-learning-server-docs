 #`is.encoding`: Test to see if supplied encoding type is supported.

 Applies to version 1.1.0 of package mrsdeploy.
 
 ##Description
 
Test to see if supplied encoding type is supported.
 
 
 ##Usage

```   
  is.encoding(type)
 
```
 
 ##Arguments

   
  
 ### `type`
 The encoding type to check for validity.
