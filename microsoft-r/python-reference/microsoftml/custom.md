## *custom*: Removes custom stop words


*Applies to:* SQL Server 2017, Machine Learning Services 9.3


### Usage



```
microsoftml.custom(stopword: list = None)
```




### Description

Remover with specified by user list of stopwords.


### Arguments


##### stopword

List of stopwords (settings).
